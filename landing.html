Instalasi

Bisa diinstal di semua perangkat Anda

Sekali diinstal, aplikasi berjalan seperti aplikasi native — dengan ikonnya sendiri, dan bisa dipakai offline.

📱 Android (Chrome)

  1. Buka link ini di Chrome
  2. Tap banner "Tambahkan ke Layar Utama" di bawah layar (atau menu ⋮ → Instal aplikasi)
  3. Tap Instal

🍎 iPhone / iPad (Safari)

  1. Buka link ini di Safari (wajib Safari)
  2. Tap ikon Share (kotak dengan panah ke atas)
  3. Tap Add to Home Screen → Add

💻 Windows (Chrome / Edge)

  1. Buka link ini di Chrome atau Edge
  2. Klik ikon Install di address bar (atau menu ⋮ → Instal aplikasi)
  3. Klik Install

🖥️ Mac (Safari / Chrome)

  1. Safari: menu File → Add to Dock...
  2. atau Chrome: ikon Install di address bar
  3. Klik Add / Install

Kenapa Anda bisa percaya aplikasi ini?

GeoPass Vault tidak punya server, tidak punya backend, dan tidak pernah mengirim data Anda kemana pun. Semua enkripsi terjadi langsung di perangkat Anda menggunakan Web Crypto API bawaan browser — teknologi standar yang sama dipakai oleh bank dan aplikasi keamanan lainnya.

Kode sumbernya terbuka sepenuhnya di GitHub — siapa pun bisa memeriksa persis apa yang dilakukan aplikasi ini, baris demi baris. Tidak ada iklan, tidak ada pelacakan, tidak ada model bisnis tersembunyi.

Penting: karena tidak ada server, tidak ada juga fitur "lupa password". Simpan Master Password Anda baik-baik, dan gunakan fitur backup secara rutin.
