Kenapa Anda bisa percaya aplikasi ini?

GeoPass Vault tidak punya server, tidak punya backend, dan tidak pernah mengirim data Anda kemana pun. Semua enkripsi terjadi langsung di perangkat Anda menggunakan Web Crypto API bawaan browser — teknologi standar yang sama dipakai oleh bank dan aplikasi keamanan lainnya.

Kode sumbernya terbuka sepenuhnya di GitHub — siapa pun bisa memeriksa persis apa yang dilakukan aplikasi ini, baris demi baris. Tidak ada iklan, tidak ada pelacakan, tidak ada model bisnis tersembunyi.

Penting: karena tidak ada server, tidak ada juga fitur "lupa password". Simpan Master Password Anda baik-baik, dan gunakan fitur backup secara rutin.
